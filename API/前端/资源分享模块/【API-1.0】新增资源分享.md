## 功能描述
```
已注册的用户发布资源分享
```

## 调用方式

| Key              | Value |
| ---------------- | ----- |
| **HTTP请求方式** | POST  |
| **支持格式**     | JSON  |
| **是否需要登录** | 否    |

## 入参说明



| 参数名     | 必选 | 类型及范围 | 描述                             |
| ---------- | ---- | ---------- | -------------------------------- |
| sesssionId | 是   | String     |                                  |
| content    | 是   | String     | 资源内容                         |
| type       | 是   | String     | 资源类型（前端人员自建下拉菜单） |

## 入参示例
```
{
	"sessionId":"CB82479072E7AFAE599C6D15CE32B211",
	"content":"这是我上传的点子书，大家可以点击下载，www.baidu.com",
	"type":"电子书资源"
}
```

---

## 返回值说明
| 参数名   | 类型及范围 | 描述     |
| -------- | ---------- | -------- |
| code     | String     | 返回代码 |
| message  | String     | 返回信息 |
| contents | String     | 返回内容 |

## 返回值示例
```
{
    "code": "200",
    "message": "插入成功"
} 
```

```
{
    "code": "500",
    "message": "参数不合法"
}
```
---

## 补充说明
~~~js
200 代表返回成功
500 代表待用失败
~~~
