## 调用方式

| ---------------- | ----- |

## 入参说明

| 参数名 | 必选 | 类型及范围 | 描述     |
| ------ | ---- | ---------- | -------- |
| uName  | 是   | String     | 用户昵称 |

## 入参示例
```js
{
	"uName":"xiuxiu"
}
```

## 返回值说明
| 参数名  | 类型及范围 | 描述     |
| ------- | ---------- | -------- |
| code    | String     | 返回代码 |
| message | String     | 返回信息 |
